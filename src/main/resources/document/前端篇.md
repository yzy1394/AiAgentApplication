# 前端篇：常见问题与回答

## 1. Vue2 与 Vue3 如何选择？
新项目优先 Vue3，生态成熟且组合式 API 更利于复杂逻辑复用；旧项目可渐进迁移。

## 2. 前后端联调如何更高效？
先定义 OpenAPI 契约，再用 Mock 数据并行开发；统一错误码与响应结构，减少联调反复。

## 3. 如何管理复杂状态？
按业务域拆分 store，避免全局状态膨胀；异步流程统一封装，保证状态更新可追踪。

## 4. 如何优化首屏加载？
做路由懒加载、静态资源压缩、关键路径预加载，避免一次性加载全部模块。

## 5. 如何保证组件可维护性？
组件职责单一，输入输出明确，避免隐式依赖；通用能力抽成 composables 或 hooks。
